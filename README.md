# Проект "Wardrobe" — Цифровой гардероб

## О проекте

**Wardrobe** — это веб-приложение, разработанное на Django, которое позволяет пользователям каталогизировать свою одежду, создавать из нее стильные образы (аутфиты) и управлять своим цифровым гардеробом. Проект был создан в рамках учебного курса для демонстрации навыков full-stack разработки.

---

## Основные возможности

* **Аутентификация пользователей:** Регистрация, вход и выход из системы.
* **Управление гардеробом:**
    * Добавление новых вещей с загрузкой фотографий.
    * Просмотр списка всех вещей с пагинацией.
    * Детальная страница для каждой вещи.
    * Редактирование и удаление вещей.
    * Динамическая фильтрация вещей по сезону на JavaScript без перезагрузки страницы.
* **Создание образов (Аутфитов):**
    * Создание образов из уже существующих в гардеробе вещей.
    * Просмотр списка созданных образов.
    * Редактирование и удаление образов.
* **API:**
    * Read-only API эндпоинт для получения списка всех вещей в формате JSON.
    * Эндпоинт для получения детальной информации об одной вещи по ее ID.
* **Админ-панель Django:** Кастомизированная панель для удобного управления всеми моделями данных.

---

## Технологии

* **Backend:** Python, Django, Django REST Framework
* **Frontend:** HTML5, CSS3, JavaScript
* **База данных:** SQLite3 (в разработке)

## Инструкция по установке и запуску

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <ссылка-на-твой-репозиторий>
    cd Wardrobe1
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    # Для Windows
    python -m venv venv
    .\venv\Scripts\activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Примените миграции базы данных:**
    ```bash
    python manage.py migrate
    ```

5.  **Создайте суперпользователя (администратора):**
    ```bash
    python manage.py createsuperuser
    ```

6.  **Запустите сервер для разработки:**
    ```bash
    python manage.py runserver
    ```

После этого проект будет доступен по адресу `http://127.0.0.1:8000/`.
